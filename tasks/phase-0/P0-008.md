## Objective

Add "View/Manage Booking", "Reschedule", and "Cancel" links to the booking confirmation email so customers can manage their appointment directly from the email.

## Acceptance Criteria

- [ ] Booking confirmation email includes a "Manage Your Booking" button/link
- [ ] Link points to `/book/manage/{bookingReference}`
- [ ] Cancellation and reschedule email templates also include the manage link
- [ ] The base URL is constructed from environment variable or request origin
- [ ] `pnpm build` passes

## Context

- **Email template**: `src/lib/email-templates.ts` — `bookingConfirmationEmail()` function (line ~180)
- **Current state**: Email shows service, staff, date/time, reference, and contact info. No action links.
- **Also update**: `appointmentRescheduledEmail()` and `appointmentCancelledEmail()` templates.
- **Callers**: `src/lib/actions/public-booking.ts` sends the confirmation email. It will need to pass the `baseUrl` and `bookingReference`.

## Implementation Notes

### Step 1: Add params to email template

```typescript
bookingConfirmationEmail({
  // existing params...
  bookingRef: string,
  manageUrl: string,  // NEW: full URL to /book/manage/{bookingReference}
})
```

### Step 2: Add manage section to email HTML

After the booking details table, add:
```html
<div style="text-align: center; margin: 24px 0;">
  <a href="{{manageUrl}}" style="background-color: #059669; color: white; padding: 12px 24px; border-radius: 6px; text-decoration: none; font-weight: 600;">
    Manage Your Booking
  </a>
</div>
<p style="text-align: center; color: #6b7280; font-size: 14px;">
  Need to reschedule or cancel? Use the link above.
</p>
```

### Step 3: Construct URL in caller

In `createPublicBooking()` action:
```typescript
const baseUrl = process.env.NEXT_PUBLIC_APP_URL || 'http://localhost:3000'
const manageUrl = `${baseUrl}/book/manage/${bookingReference}`
```

Pass `manageUrl` to `bookingConfirmationEmail()`.

### Step 4: Update reschedule/cancel email templates

Add the same manage link to `appointmentRescheduledEmail()` and `appointmentCancelledEmail()` for consistency.

pnpm build
