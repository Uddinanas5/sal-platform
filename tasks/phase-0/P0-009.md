## Objective

Replace hardcoded demo URLs in the BookingQRCode and BookingWidgetCode components with real URLs based on the current business's slug.

## Acceptance Criteria

- [ ] BookingQRCode displays and copies the real booking URL: `{baseUrl}/book/{businessSlug}`
- [ ] BookingWidgetCode generates iframe and button embed snippets with the real URL
- [ ] QR code SVG encodes the real URL (or at minimum, the displayed URL matches the real one)
- [ ] Both components accept `businessSlug` as a prop
- [ ] Download QR code button works (or is removed if not feasible)
- [ ] `pnpm build` passes

## Context

- **BookingQRCode**: `src/components/booking/booking-qr-code.tsx` — hardcodes `"https://book.meetsal.com/luxe-salon"`. QR SVG is a static 21x21 pattern (not dynamically generated).
- **BookingWidgetCode**: `src/components/booking/booking-widget-code.tsx` — hardcodes `"https://meetsal.ai/book/sal-salon"` in iframe and button snippets.
- **Dashboard booking client**: `src/app/(dashboard)/booking/client.tsx` renders both components. It may need to pass the slug down.
- **Business slug**: Available from the session/business context in the server page.

## Implementation Notes

### Step 1: Accept businessSlug prop

Both components need a `businessSlug` prop:
```typescript
interface BookingQRCodeProps {
  businessSlug: string
}
```

### Step 2: Construct real URL

```typescript
const baseUrl = process.env.NEXT_PUBLIC_APP_URL || window.location.origin
const bookingUrl = `${baseUrl}/book/${businessSlug}`
```

### Step 3: Fix QR code

The current QR SVG is a hardcoded static pattern. Options:
- **Option A (simple)**: Keep the static QR SVG as a decorative element but ensure the displayed/copied URL is correct. The QR pattern won't encode the real URL but the copy button works.
- **Option B (proper)**: Use a QR generation library (e.g., `qrcode` npm package) to generate a real QR SVG from the booking URL. This is better but adds a dependency.

Recommend Option A for now (scope control), with a comment noting the QR pattern is decorative.

### Step 4: Fix widget code

Replace hardcoded URLs in the iframe and button code templates:
```typescript
const iframeCode = `<iframe src="${bookingUrl}" width="100%" height="700" ...`
const buttonCode = `<a href="${bookingUrl}" style="...">Book Now</a>`
```

### Step 5: Pass slug from server page

In `src/app/(dashboard)/booking/page.tsx`, fetch the business slug and pass it through to the client components.
